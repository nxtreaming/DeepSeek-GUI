# Kun Video Editor UI reference set

These ImageGen mockups are the visual contract for the right-sidebar editor. They
share one compact creator-tool language and map directly to the six primary UI
states implemented by the Webview:

| State | Reference |
| --- | --- |
| Initial project setup | `00-initial-project.png` |
| Script and transcript | `01-script-workspace.png` |
| Clips and media library | `02-clips-workspace.png` |
| Spatial timeline | `03-timeline-workspace.png` |
| Selected-item properties | `04-properties-workspace.png` |
| Output and delivery | `05-output-workspace.png` |

The mockups define hierarchy, density, grouping, color, and responsive intent.
Runtime UI remains driven by real project capabilities and never fabricates media,
proof, analysis, or job state merely to match a visual.
